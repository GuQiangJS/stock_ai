stock_ai.data_processor.load_index_daily(code, **kwargs)[源代码]¶

读取指数日线数据（在线或从数据库）

参数

  • code (str) – 指数代码。

  • start (str, optional) – 开始日期。数据格式为%Y-%m-%d。默认为`1990-01-01`。

  • end (str, optional) – 结束日期。数据格式为%Y-%m-%d。默认为`当天`。

  • online (bool, optional) – 是否获取在线数据。默认为 False。

  • columns (tuple, optional) – 获取的列集合。默认为 [‘open’, ‘high’, ‘low’, ‘close’, ‘up_count’, ‘down_count’, ‘volume’,’amount’]

返回

股票日线数据。

返回类型

pandas.DataFrame

stock_ai.data_processor.load_stock_daily(code, **kwargs)[源代码]¶

读取股票日线数据（在线或从数据库）

参数

  • code (str) – 股票代码。

  • start (str, optional) – 开始日期。数据格式为%Y-%m-%d。默认为`1990-01-01`。

  • end (str, optional) – 结束日期。数据格式为%Y-%m-%d。默认为`当天`。

  • online (bool, optional) – 是否获取在线数据。默认为 False。

  • fq (str, optional) – 是否取复权数据。默认为 qfq。 * qfq - 前复权 * hfq - 后复权 * bfq or None - 不复权

  • columns (tuple, optional) – 获取的列集合。默认为 [‘open’, ‘high’, ‘low’, ‘close’, ‘volume’, ‘amount’]

返回

股票日线数据。

返回类型

pandas.DataFrame
