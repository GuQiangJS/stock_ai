stock_ai.calcs.is_trade_suspension(df, **kwargs)[源代码]¶

返回是否为停牌

根据 df 中指定参数 column 找到指定列，采用 pandas.Series.isna() 判断。 True 表示是停牌日。

参见

pandas.Series.isna()

参数

  • df (DataFrame) – 原始数据。

  • column (str) – 用来判断的列名。默认为 close。

Examples

>>> import pandas as pd
>>> from stock_ai import calcs
>>> df = pd.DataFrame({'A':[1,2,None,4,None]})
>>> calcs.is_trade_suspension(df, column='A').values
array([0., 0., 1., 0., 1.])
返回

如果是停牌数据则为1，否则为0。

返回类型

pandas.Series
