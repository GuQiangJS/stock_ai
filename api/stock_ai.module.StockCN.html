StockCN.get_daily(start=None, end=None)[源代码]¶

获取日线数据。

当前实例会缓存所有的日线数据。再从缓存中提取指定日期的数据作为返回值。

参数

  • start (str) – 开始日期。

  • end (str) – 结束日期。

返回

日线数据。 如果 start 和 end 相同，则返回一日数据。 数据参考 stock_ai.data_processor.load_stock_daily()

返回类型

DataFrame

StockCN.get_sharpe_ratio(**kwargs)[源代码]¶

计算指定时间段的夏普比率。

参数

  • start (str or datetime.datetime) – 开始时间。参考 get_daily()。

  • end (str or datetime.datetime) – 开始时间。参考 get_daily()。

:param 如果传入字符串，需要满足 stock_ai.util.str2date() 方法要求。: :param rf: 无风险利率。默认为 1.0。 :type rf: float :param split: 分割规则。包含 year,none。none为不分割。默认为 year。 :type split: str :param column: 计算列。默认为 close。 :type column: str
