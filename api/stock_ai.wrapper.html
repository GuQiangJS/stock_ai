stock_ai.wrapper.dataframe_merge(df1, df2, **kwargs)[源代码]¶

股票日线数据与指数日线数据合并

参数

  • df1 (DataFrame) – 股票数据。

  • df2 (DataFrame) – 指数数据。

  • copy (bool) – 是否使用copy后的数据。参考 pandas.DataFrame.copy() 方法。默认为 True。

  • rsuffix (str) – 默认为 _index。参考 pandas.DataFrame.join() 中同名参数。

  • how (str) – 默认为 right。参考 pandas.DataFrame.join() 中同名参数。

  • append_funcs (collections.OrderedDict) –

    附加其他列数据时的计算方法字典。

    • key值为需要附加的列名。

    • value值为可以为方法名称或者是[方法名,参数字典](详见 stock_ai.appender)。

    关于附加列名的赋值方式：

    • 如果方法返回的是 pandas.Series，用key值来赋值。

    • 如果方法返回的是 pandas.DataFrame，会用返回的 DataFrame 中的列名+Key值赋值。

参见

pandas.DataFrame.join()

Examples

使用默认参数合并。

>>> df_1 = data_processor.load_stock_daily('601398')
>>> df_2 = data_processor.load_index_daily('399300')
>>> wrapper.dataframe_merge(df_1, df_2).head()
.            open  high  low  ...  down_count  volume_index  amount_index

附加列只传递方法名（无参数）

>>> funcs = {'year': stock_ai.appender.calc_year}
>>> df = wrapper.dataframe_merge(df_1, df_2, append_funcs=funcs)
>>> df['year'].head()
date

2005-01-05    2005
2005-01-06    2005
2005-01-07    2005

Name: year, dtype: int64

附加列时传递参数的方式

>>> def app(df, **kwargs):
>>>    v = kwargs.pop('v', '123')
>>>    return pd.Series(v, index=df.index)
>>> new_col = 'cv'
>>> new_value='123'
>>> funcs={new_col: [app, {'v': new_value}]}
>>> df = wrapper.dataframe_merge(df_1, df_2, append_funcs=funcs)
>>> df[new_col].head()
